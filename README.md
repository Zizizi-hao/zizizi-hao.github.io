# zihao's Personal Website

> Hi~ I am zihao, a Ph.D. and a PM for myself.

This repository hosts my personal academic website, built with [Academic Pages](https://academicpages.github.io/) — a ready-to-fork GitHub Pages template designed for academics and researchers.

## About This Site

This website showcases:
- **Publications** – Research papers and academic work
- **Projects** – Side projects and technical explorations
- **Blog** – Thoughts on research, technology, and productivity
- **CV** – Dynamically generated from structured data

## Tech Stack

| Component | Technology |
|-----------|------------|
| Static Site Generator | [Jekyll](https://jekyllrb.com/) |
| Hosting | [GitHub Pages](https://pages.github.com/) |
| Theme | [Academic Pages](https://academicpages.github.io/) |
| Markdown | Content authoring |

## Local Development

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```
